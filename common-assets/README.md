# Common Assets

Stock assets shared across all site builds. Drop files into the appropriate folder.

- `textures/` — marble, wood, concrete, fabric backgrounds
- `patterns/` — subtle repeating patterns, geometric designs
- `icons/` — service icons (scissors, tooth, dumbbell, gavel, etc.)
- `backgrounds/` — hero overlays, gradients, abstract shapes
- `placeholders/` — generic team silhouettes, placeholder product shots

All assets here should be copyright-free (Unsplash, Pexels, free icon sets).
When building a site, Kiro can copy needed assets from here into the project.
